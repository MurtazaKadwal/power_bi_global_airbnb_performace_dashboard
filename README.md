# Global Airbnb Performance Dashboard

## 📌 Project Overview
This project presents an interactive **Power BI dashboard** built to analyze global Airbnb performance trends, customer satisfaction metrics, pricing insights, and market distribution across major cities worldwide.

The dashboard transforms raw Airbnb data into actionable insights using advanced visualizations and business intelligence techniques.

---

## 🎯 Objectives
- Analyze Airbnb listing growth trends over the years
- Compare performance across different cities
- Understand pricing variations among room types
- Evaluate customer review and satisfaction metrics
- Identify top-performing cities and market leaders
- Build a clean and interactive business dashboard

---

## 📊 Dashboard Highlights

### 1. Global Performance KPIs
The dashboard tracks:
- Total Listings
- Number of Cities
- Hosts
- Property Types
- Reviews

### 2. Listing Trends Over Time
Visualizes Airbnb growth trends from 2008–2020 and highlights:
- Peak growth years
- Impact of regulations
- COVID-19 slowdown effect

### 3. Market Share by City
Compares city-level contribution to:
- Listings
- Reviews
- Superhost distribution

### 4. Pricing Analysis
Analyzes average room prices for:
- Hotel Room
- Entire Place
- Shared Room
- Private Room

### 5. Customer Satisfaction Analysis
City-wise comparison based on:
- Accuracy
- Communication
- Cleanliness
- Value
- Location

---

## 🔍 Key Insights
- Paris, NYC, and Sydney contribute nearly half of Airbnb listings and reviews.
- Paris has the highest listings and review counts.
- Hotel rooms have the highest average prices.
- Mexico City and Rio de Janeiro received the best overall ratings.
- Cleanliness and value-for-money were the lowest-rated metrics overall.
